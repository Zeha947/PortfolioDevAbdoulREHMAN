TP3

Responsive Design & Media Queries

Analyse de l'Exercice 1 : Rectangles Adaptatifs

Connaissances mises en jeu : L'exercice est une introduction fondamentale aux Media Queries. Il s'agit de modifier les propriétés d'un élément (ici la couleur de fond) en fonction de la largeur du viewport. On utilise les points de rupture (breakpoints) à 960px et 480px.

Difficulté : Très Facile Si la condition de largeur est remplie, le style s'applique.

Intérêt pédagogique : L'exercice permet de comprendre comment le CSS réagit dynamiquement à la taille de l'écran est le premier pas vers l'accessibilité mobile.

Qualité du résultat : Le code est bon et parfaitement fonctionnel. Le passage du bleu au jaune, puis au rouge, permet de valider instantanément le bon fonctionnement des règles CSS.

Analyse de l'Exercice 2 : Mise en page Mobile-Ready

Connaissances mises en jeu : Ici, le Responsive Design est appliqué à une structure complète. On combine CSS Grid pour la grille principale et le footer, avec des Media Queries pour réorganiser le layout. Le passage de 3 colonnes à 2, puis à 1 seule colonne sur mobile, est l'aspect central de l'exercice.

Difficulté : Facile. La structure Grid simplifie la réorganisation des éléments par rapport aux anciennes méthodes (floats).

Intérêt pédagogique : C'est un cas concret de mise en page "desktop-first". L'exercice permet de pratiquer la gestion des espacements (`gap`) et des arrondis de bordure de manière cohérente sur l'ensemble du document.

Qualité du résultat : Le code est propre et bien organisé. L'utilisation de variables de couleurs (cyan, magenta) et de classes réutilisables (`.box`) rend le code maintenable. L'aspect fonctionnel est impeccable sur tous les formats d'écran.
